# 六自由度机械臂模型说明

## 模型概述
本项目当前采用一套更适合三维空间轨迹跟踪的通用 6R 机械臂结构，使用 Modified DH 参数建立串联运动学模型。结构划分如下：

- `J1`：底座回转
- `J2`：肩关节俯仰
- `J3`：肘关节俯仰
- `J4`：腕部回转
- `J5`：腕部俯仰
- `J6`：工具法兰回转

该模型的目标是兼顾三点：

- 参数足够完整，可直接用于 Robotics Toolbox 和 Python 代码
- 工作空间相对更大，便于演示斜平面圆、空间螺旋线和一般 3D 轨迹
- 保留前三轴定位置、后三轴定姿态的典型 6R 教学结构

## MDH 参数表
单位约定：

- 长度：`m`
- 角度：`deg`

| 关节 | 名称 | a | alpha | d | offset | 角度范围 Q0 ~ Q1 |
| --- | --- | ---: | ---: | ---: | ---: | ---: |
| J1 | base_yaw | 0.00 | 90 | 0.18 | 0 | -170 ~ 170 |
| J2 | shoulder_pitch | 0.32 | 0 | 0.00 | 0 | -100 ~ 100 |
| J3 | elbow_pitch | 0.28 | 0 | 0.00 | 0 | -135 ~ 135 |
| J4 | wrist_yaw | 0.00 | 90 | 0.12 | 0 | -180 ~ 180 |
| J5 | wrist_pitch | 0.00 | -90 | 0.00 | 0 | -125 ~ 125 |
| J6 | tool_roll | 0.00 | 0 | 0.10 | 0 | -360 ~ 360 |

额外工具变换：

- 工具平移：`[0, 0, 0] m`
- 工具姿态：`[0, 0, 0] deg`

## offset 设计理由
当前项目默认仍然使用全零 offset，以保持数学模型简洁，并方便 MATLAB、Python、Qt 的一致性验证：

- `J1 ~ J6` 的 `offset_deg` 统一为 `0`
- 默认零位更容易和 MDH 推导、雅可比分析以及逆解调试对应
- 如果后续你拿到真实装配零位，再单独补 offset 即可

## 默认初始条件
- 默认初始关节角：`[41.37, -35.43, 47.2, 43.55, 49.29, 7.18] deg`
- 默认目标关节角：`[20, -28, 58, 25, 38, 25] deg`
- 当前默认初始姿态按“末端更靠近稳定工作空间、位置雅可比条件数更低、腕部远离奇异区”的原则选取。
- 默认仿真时长：`6 s`
- 默认采样点数：`181`
- 默认空间曲线参数：
  - 半径：`0.04 m`
  - 轴向抬升：`0.12 m`
  - 螺旋圈数：`1.5`
  - 轴方向：`[1, 1, 1]`

## 坐标与建模约定
- 采用 `Modified DH`
- 所有关节均为转动关节
- MATLAB 与 Python 都从同一份 `config/robot_mdh_template.json` 读取参数
- `offset` 在求解中直接叠加到关节变量 `theta`
- 末端位姿输出包括：
  - 齐次变换矩阵 `T`
  - 位置 `x, y, z`
  - 欧拉角 `roll, pitch, yaw`

## 结果输出约定
统一输出遵循 `config/result_schema.json`：

- `summary`：数据来源、轨迹类型、采样长度
- `joint_trajectory`：每个采样点的关节角度
- `pose_trajectory`：每个采样点的末端位置与姿态
- `metrics`：轨迹误差等指标

## 后续替换真实机械臂时的操作
如果后续你拿到今年新机械手的尺寸和零位，只需要替换以下内容：

1. 更新 `config/robot_mdh_template.json` 中的 `a`、`alpha_deg`、`d`
2. 更新每个关节的 `offset_deg` 和 `qlim_deg`
3. 保持 MATLAB、Python、Qt 代码不变，重新运行仿真即可
